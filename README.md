🚀 SQL Server Job Monitoring & Alerting System

📌 Overview
This project demonstrates a production-ready SQL Server DBA system that includes automated backup, job monitoring, and alerting mechanisms.

---

🛠 Features

🔹 Backup Automation
- Full Backup (Daily - 3 AM)
- Dynamic file naming
- Backup retention policy (7 days cleanup)

🔹 Job Monitoring
- Detect failed jobs
- Detect long-running jobs
- Real-time monitoring using MSDB tables

🔹 Alert System
- Email notifications using Database Mail
- Failure threshold-based alerting
- Alert logging to avoid spam

---

🧠 Concepts Covered
- SQL Server Agent Jobs
- MSDB system tables (sysjobs, sysjobhistory, sysjobactivity)
- Backup & Restore Strategy
- Automation using T-SQL
- Alerting & Monitoring Systems

---

⚙️ Tools Used
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

---

📂 Project Structure
- Database setup scripts
- Backup automation scripts
- Monitoring stored procedures
- Alert configuration scripts
- Job scheduling scripts

---

Successfully built a real-world DBA monitoring and backup automation system that proactively detects failures and ensures data safety.
